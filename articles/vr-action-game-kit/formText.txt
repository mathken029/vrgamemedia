【プレスリリース】未経験者を後押しする無料VRゲーム開発キット『VR Action Game Kit』を発表

いつも大変お世話になっております。
個人ゲーム開発者の@_mathkenと申します。

さて、今回私は2023年06月27日、VRゲーム開発未経験者を後押しする、
無料VRゲーム開発キット『VR Action Game Kit』を発表しました。
以下のプレスリリースの概要を記載します。

・プレスリリースページ
https://vrgamemedia.com/2023/06/27/vr-action-game-kit-press/

『VR Action Game Kit』は、VRでアクションゲームを作る際に必要な近接武器や
ダメージのシステムを備えたゲーム開発キットです。
オープンソースで提供し、商用利用を含めて自由に活用いただけます。

機能の概要がわかるサンプルゲームのプレイ動画を以下からご覧いただけます。

・サンプルゲームプレイ動画
https://www.youtube.com/watch?v=4ZCW1B1Xkrs


本キットは以下の特長を備えています。

・Unity社公式アセットを活用し、基本的な近接武器や敵、エフェクトやステージ・ギミックなどを備えています
・コントローラーで移動・視点変更した際に視界を狭くすることでVR酔い対策を実装しています
・本キットにはモンスターとロボットの敵が登場し、ヒットポイントや攻撃力を設定したり攻防が行なえます

本ツールは以下にて公開中です。使用方法の説明ページもございます。

https://github.com/mathken029/vr-action-game-kit
